# 🎯 Digital Tools Buying Website

## 📖 Description
This is a modern digital tools buying platform where users can explore various digital products, add them to the cart, and manage purchases easily. The project focuses on clean UI, smooth user experience, and efficient cart management system.

## ⚙️ Technologies Used
- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6)
- React Toastify

## 🚀 Features
- 🛒 Add to cart with real-time update
- ❌ Remove items from cart easily
- 💰 Dynamic total price calculation
